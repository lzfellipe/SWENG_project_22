# CSU2013/3013 SWENG Group 22
## Bubble Vision

Bubble Vision (originally Fisconnect) is a platform for connecting SMEs to visual content creators.

This project involved building the platform using ReactJS.

![Bubble Vision Homepage](https://media.giphy.com/media/M6j7ZgTgSD7Z2a8DMJ/giphy.gif)

SMEs can browse freelancers listed on the platform.

![Browse Freelancers](https://media.giphy.com/media/1BXwvGwWXf7k8L7pPE/giphy.gif)

Freelancers have their own dedicated portfolio page on the platform.

![Freelancer Portfolio Page](https://media.giphy.com/media/a5aT2FECTv39bxbdrB/giphy.gif)


**A full walk through of all the use cases can be found [here](https://www.youtube.com/watch?v=zRCAOKVXYcg) or in the deployment file.**
